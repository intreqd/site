### Install

#### Claude
Works out of the box with Claude Code

```bash
# Install all agents to your Claude Code directory
./scripts/install.sh --tool claude-code
```

#### Antigravity (Others)
Need a convert step for others.

```bash
# Step 1 -- generate integration files for all supported tools
./scripts/convert.sh 

# Step 2 -- install interactively (auto-detects what you have installed)
./scripts/install.sh --tool antigravity
./scripts/install.sh --tool gemini-cli
```

### Certified Agent Factory

This is how the agents were built (or the sausauge was rolled), feel free to use this on any pattern where the curriculum is available for your RAG call and build specific agents tailored to that certification. I got this done with a fork of the great [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents) and the below prompts.

### InterSystems HL7 Interface Specialist
[InterSystems HL7 Interface Specialist Certification](https://www.intersystems.com/certifications/intersystems-hl7-interface-specialist/)

```text
activate engineering technical writer and please re-write the intersystems-hl7-interface-specialist.md file to be slightly more technical and include the skills at https://www.intersystems.com/certifications/intersystems-hl7-interface-specialist/ provide basic understanding of hl7 v2 messaging, fhir, and ccda, ideally this agent should act as a technical bridge to implementing hl7 interfaces or fhir solutions and transforming messages to various formats to build solutions.
```

### InterSystems IRIS Development Professional
[InterSystems IRIS Development Professional Certification](https://www.intersystems.com/certifications/intersystems-iris-development-professional/)

```text
activate engineering technical writer and rewrite intersystems-iris-development-professional.md to be completely inline with the skill set involved with this certification track https://www.intersystems.com/certifications/intersystems-iris-development-professional/ please be exceptionally inclusive of intersystems objectscript programming and do not update the structure of the file, just the content
```

### InterSystems IRIS SQL Specialist
[InterSystems IRIS SQL Specialist Certification](https://www.intersystems.com/certifications/intersystems-iris-sql-specialist/)

```text
Please ONLY update and consider intersystems-iris-sql-specialist.md and leave the structure intact, its ok to add examples of dynamic sql or inline sql (be objectscript and sql specific), use the "template-sql" for some hints and also https://www.intersystems.com/certifications/intersystems-iris-sql-specialist/ to build the agent as this is the role and some skills needed to be fulfilled by the agent. Please lean on the technical side and cover SQL, Relational Tables, Indexing, Query Plans
```

### InterSystems IRIS System Administration Specialist
[InterSystems IRIS System Administration Specialist Certification](https://www.intersystems.com/certifications/intersystems-iris-system-administration-specialist/)

```text
update intersystems-iris-system-administrator.md and ONLY consider this contents for the update. We are building an agent that carries out tasks around the InterSystems IRIS/Cache database specific to the skills outlined here: https://www.intersystems.com/certifications/intersystems-iris-system-administration-specialist/ again, please keep the structure intact and just update the content, this is heavy on system administration , System Diagnostics, Backup, High Availability, System Performance, System Security, Mirroring
```
